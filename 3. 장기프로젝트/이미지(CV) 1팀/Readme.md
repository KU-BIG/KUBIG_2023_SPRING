# 웹툰 이미지 가상 캐스팅

> ## 프로젝트 목표
웹툰 이미지 기반 배우 **가상 캐스팅** 모델 구현
- input으로 웹툰 이미지를 넣으면 output으로 닮은 배우를 추천해주는 모델 구현

> ## 프로젝트 소개
- **기간** : 2023.03~2023.07 
- **팀원**
  **16기** 임채명 윤지현
  **17기** 이서연 황우현
